# ZQM Security Policy Samples

Sample policy documents aligned with the ZQM attestation toolkit control set.

Contents

- `sample_endpoint_security_policy.md` — baseline Windows endpoint policy for enterprise/SOC adoption
- `sample_change_control_policy.md` — firmware/OS/package change policy
- `sample_incident_response_runbook.md` — abbreviated IR playbook tied to attestation evidence

Intended use

- Procurement reviewers evaluating policy alignment
- MSPs comparing control frameworks
- Customers requesting policy templates as part of compliance review

These docs are intentionally generic. They contain no organization-specific attributes, internal host references, or credentials.
